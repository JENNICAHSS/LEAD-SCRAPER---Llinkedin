# Lead Scraper — Chrome Extension

Free LinkedIn lead scraper. Type any search, get a CSV with
names, phone numbers, emails, addresses, hours, and website links.

---

## How to Install (takes 30 seconds)

1. **Unzip** this folder somewhere on your computer

2. Open Chrome and go to:
   chrome://extensions/

3. Turn ON **Developer mode** (top-right corner toggle)

4. Click **"Load unpacked"**

5. Select the `lead_scraper_extension` folder

6. The extension icon appears in your toolbar ✅

---

## How to Use

1. Click the extension icon in Chrome
2. Toggle the switch **ON**
3. Type your search — examples:
   - medical spa owners North Carolina
   - realtors in Austin Texas
   - gym owners Florida
   - dentists Phoenix Arizona
   - restaurant owners Chicago Illinois
4. Choose how many pages (more pages = more leads, takes longer)
5. Click **▶ Scrape Leads**
6. Watch the live log — it opens background tabs automatically
7. A CSV file downloads when done — opens in Excel

---

## What You Get (per lead)

- Full Name
- Job Title
- Company
- Address
- Phone Number
- Business Hours
- Rating & Reviews
- Website URL
- Email Address
- LinkedIn Profile URL
- Google Maps Link
- Bio / Summary

---

## Notes

- The extension opens brief background tabs on Google — this is normal
- Built-in delays prevent Google rate limiting
- 30 leads takes roughly 5-8 minutes
- Turn the toggle OFF when you're done — browser works normally
- Output is a .csv file — opens directly in Excel or Google Sheets

---

## Troubleshooting

**"No results found"**
→ Try simpler keywords: "dentist North Carolina" instead of "dentists in NC"

**Extension not showing up**
→ Make sure Developer mode is ON in chrome://extensions/

**Google shows CAPTCHA**
→ Wait 10-15 minutes and try again with fewer pages
