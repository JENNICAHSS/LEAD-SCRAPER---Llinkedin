Install in 4 steps
1

Download & extract

Click the Download button above. Once downloaded, right-click the zip → Extract All (Windows) or double-click (Mac).

2

Open Chrome Extensions

In Chrome, type chrome://extensions/ in the address bar and press Enter.

3

Enable Developer Mode

Toggle Developer mode ON in the top-right corner of the Extensions page. Three buttons will appear.

4

Load the extension

Click "Load unpacked" → select the extracted lead_scraper_extension folder → click Open. Done!
